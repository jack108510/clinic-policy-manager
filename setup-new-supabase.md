# Setting Up a New Supabase Project

## Step 1: Create the Project

1. Go to: https://supabase.com/dashboard
2. Click **"New Project"**
3. Fill in:
   - **Name**: `Inventory Ordering Platform` (or any name you prefer)
   - **Database Password**: Choose a strong password (save it!)
   - **Region**: Choose the closest to you
4. Click **"Create new project"**
5. Wait 2-3 minutes for it to finish setting up

## Step 2: Get Your Project Details

Once created, you'll need:

1. **Project URL**: 
   - Go to: Settings → API
   - Copy the "Project URL" (looks like: `https://xxxxx.supabase.co`)

2. **Anon/Public Key**:
   - Same page (Settings → API)
   - Copy the "anon public" key

3. **Service Role Key** (for admin operations):
   - Same page
   - Copy the "service_role" key (keep this secret!)

## Step 3: Tell Me Your Details

Once you have them, I'll:
- Update all configuration files
- Set up the database schema
- Create your user account
- Disable email confirmation
- Import products
